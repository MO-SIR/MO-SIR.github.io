pip的基本使用

十一月 15, 2021

​
pip是python用来安装第三方库的一个工具，简单说就是一个python专用的下载工具，常用的命令有安装和卸载，比如

`#普通安装，一般默认安装目标库的最高版本

pip install opencv`

`#指定版本安装

pip install opencv==4.1.5`

`#卸载某个第三方库

pip uninstall opencv`

`#显示已安装的第三方库

pip list`

`#或者

pip freeze`

`#查看库的详细信息，可查看库的版本及作者等信息

pip show numpy`

`#显示库中的文件信息

pip show -f numpy`

`#更新库的版本

pip install -U numpy`

pip也可以转移库的文件路径，比如，你换了个版本的python，原先的版本卸载了，但是你不想重新安装一遍第三方库，那么，在卸载旧版本前，可以先将第三方库导出到一个位置，安装了新版本后再把库导入新版本

`#先将第三方库导出，输入下面的命令

pip freeze >D:\out.txt`

`#等安装新版本成功后，在命令行窗口输入

pip install -r >D:\out.txt `

这样就不会因为版本的更换而麻烦地再安装一遍第三方库了

有时候用pip下载第三方库会比较慢，这时候可以考虑换源安装，换源的方法有两种，一种是临时换源，一种是永久换源

（一）临时换源

`#在原安装命令的基础上加上 -i+换源地址就可以了，比如用豆瓣的镜像源

pip install opencv -i http://pypi.douban.com/simple/`

（二）永久换源

1.点击此电脑，在输入框输入%APPDATA%,回车

%APPDATA%

​​​​

 在转到的路径下新建一个文件夹，命名为pip，在此文件夹内新建一个pip.ini文件

在这个文件里面输入以下内容

`
[global]
timeout = 6000 #设置超时时间，可以自行更改

index-uel = http://pypi.tuna.tsinghua.edu.cn/simple #设置默认下载源为清华大学镜像

trusted-host = pypi.tuna.tsinghua.edu.cn`

保存，这样以后使用pip命令安装第三方库的时候会默认清华大学镜像源，就不会很卡
